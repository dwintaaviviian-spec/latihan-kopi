# Latihan Kopi

Project website statis bertema kopi menggunakan HTML, CSS, dan JavaScript.  
Digunakan sebagai latihan dasar front-end dan upload project ke GitHub.

## Deskripsi
Website terdiri dari beberapa halaman:
- Beranda
- Produk
- Galeri
- Tentang
- Kontak
- Webinar

Project ini tidak menggunakan framework dan tidak memiliki backend.
